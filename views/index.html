Request Header Parser Microservice

A request header parser microservice. It returns IP address, preferred language and software of the client at the endpoint [base_url]/api/whoami.

Usage example:

Input: https://reqheader-parser.vercel.app/api/whoami

Output: { ipaddress: '189.43.5.56', language: 'en-US,en;q=0.5', software: 'Mozilla/5.0 (Macintosh; Intel Mac OS X 10_13_6) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/99.0.4844.15 Safari/537.36'}
